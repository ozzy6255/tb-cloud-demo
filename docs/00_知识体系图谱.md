# TBAnti 知识体系图谱 (Knowledge Map)

**说明**: 本文档索引了项目过程中涉及的所有核心知识点。详细内容请参考 `learning_plan.md`。

## 🗺️ 核心概念索引

### 1. 智能体与工程 (Agent Engineering)
- **Spec-Driven Development**: 为什么要在写代码前先写文档？(见 `learning_plan.md` 第1章)
- **Agent Workflow**: 规划 -> 执行 -> 验证 的循环。

### 2. 基础设施 (Infrastructure)
- **Localhost 悖论**: 为什么 AI 连不上我的电脑？(见 `learning_plan.md` 第6章)
- **Docker**: 为什么我们需要容器来运行数据库？
- **NAT & Tunneling**: 内网穿透的基本原理。

### 3. 全栈开发 (Full Stack)
- **Backend (API)**:
    - **FastAPI**: 现代 Python Web 框架。
    - **REST API**: URL 资源设计风格。
    - **ORM**: 不用写 SQL 也能操作数据库 (SQLAlchemy)。
- **Frontend (Web)**:
    - **Fetch API**: 浏览器如何向服务器要数据。
    - **CORS**: 浏览器的安全同源策略与后端中间件配置。

### 4. 数据库 (Database)
- **Connection Strings**: 它是如何包含 IP、用户、密码的。
- **SQL Server**: 微软的关系型数据库。
